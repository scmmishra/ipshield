IPShield Documentation

IPShield is a DNS server that checks IP addresses against the Firehol level 1 list and known datacenter IP ranges. This list is updated every 12 hours.

Usage

To check an IP address, send a DNS query to the IPShield server:

dig @your-server-ip ip-to-check +short

Response Types

  • SAFE: The IP is not on any blocklist.
  • DATACENTER: The IP belongs to a known datacenter.
  • FLAGGED: The IP is on the Firehol level 1 blocklist.

Example

To check the IP 8.8.8.8:

dig @your-server-ip 8.8.8.8 +short
